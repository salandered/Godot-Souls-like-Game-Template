README

charge_asset_library-materials.zip contains the following files:
    • blender_assets.cats.txt (Asset Catalog Definition file for Blender)
    • charge_asset_library-materials.blend

Requirements and notes:
    • tested and working with Blender 4.1
    • use the 'store attribute' geometry nodes modifier to add more or less wear
    • read the descriptions in the asset browser for additional information
      
How to use:
1. download and unzip `charge_asset_library-materials.zip`
2. start Blender and go to edit>preference>file paths>asset libraries
3. hit the plus icon and navigate to the unzipped charge asset library folder and confirm
4. create an asset browser window and select in the top left corner ‘charge_asset_library-materials’ (it takes a second to load the thumbnails)
5. make sure Eevee is selected as the current render engine
6. have fun!
